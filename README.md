# 📖 Descripción

SMPES es un plugin de Minecraft desarrollado desde cero para ofrecer una experiencia de supervivencia completamente personalizada.

El proyecto incorpora sistemas exclusivos como clases, armas legendarias, objetos personalizados, clanes y muchas otras mecánicas diseñadas específicamente para el servidor.

> ⚠️ **Actualmente el código fuente no está disponible.**
>
> Será publicado cuando finalice la **Temporada 1**.
