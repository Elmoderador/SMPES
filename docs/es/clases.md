# 📖 Introducción

## ¿Qué son las clases?

Las clases representan la principal forma de personalizar el estilo de juego de cada jugador dentro de SMPES.

Cada una ha sido diseñada para ofrecer una experiencia diferente, obligando al jugador a adaptar su estrategia según sus fortalezas y debilidades.

El objetivo del sistema no es crear clases "mejores" que otras, sino ofrecer alternativas equilibradas para distintos tipos de jugadores.

---

# 🎯 Filosofía del sistema

El diseño de las clases sigue varios principios fundamentales.

- ⚖️ Ninguna clase debe ser superior al resto.
- 🎮 Cada clase debe sentirse única.
- 🧠 Las decisiones del jugador deben importar.
- 🤝 Todas las clases deben poder convivir en el mismo servidor sin romper el equilibrio.
- 📈 La progresión recompensa la experiencia sin convertir a un jugador en invencible.

---

# ⚖️ Balance

Cada clase dispone tanto de ventajas como de limitaciones.

Esto evita que exista una "mejor clase" y obliga al jugador a escoger aquella que mejor se adapte a su forma de jugar.

| Característica | Objetivo |
|----------------|----------|
| ✅ Ventajas | Definir el rol de la clase. |
| ❌ Desventajas | Mantener el equilibrio general. |
| 📈 Progresión | Recompensar el tiempo invertido. |
| ✨ Habilidades pasivas | Diferenciar cada estilo de juego. |

> 💡 **Nota**
>
> El equilibrio de las clases puede modificarse entre temporadas si se detectan problemas durante las pruebas del servidor.

---

# 🏆 Progresión

Cada clase cuenta con un sistema de progresión propio.

A medida que el jugador completa los objetivos establecidos para su clase irá desbloqueando nuevas mejoras y habilidades.

La progresión está diseñada para recompensar la dedicación del jugador sin eliminar la importancia de la habilidad individual.

---

# ⚙️ Funcionamiento del Sistema

El Sistema de Clases está diseñado para ofrecer una experiencia de juego variada sin alterar la esencia de Minecraft.

Cada clase representa un estilo de juego distinto y obliga al jugador a tomar decisiones estratégicas tanto dentro como fuera del combate.

Una vez elegida una clase, el jugador desarrollará su progresión mediante niveles y desbloqueará nuevas ventajas conforme avance la temporada.

---

# 📈 Sistema de Progresión

Todas las clases comparten el mismo sistema de progresión.

Cada una dispone de varios niveles que permiten desbloquear nuevas habilidades y mejorar las ya existentes.

## Estructura

| Nivel | Estado |
|:------:|--------|
| Nivel I | Disponible |
| Nivel II | Disponible |
| Nivel III | Disponible |

> 💡 **Nota**
>
> Aunque todas las clases utilizan el mismo sistema de progresión, las habilidades desbloqueadas son completamente diferentes entre ellas.

---

# ✨ Habilidades Pasivas

Cada clase dispone de habilidades pasivas exclusivas.

Estas habilidades permanecen activas de forma permanente mientras el jugador utiliza dicha clase y forman parte de su identidad.

Las habilidades pasivas están diseñadas para:

- 🎮 Definir el estilo de juego.
- ⚔️ Diferenciar cada clase.
- ⚖️ Mantener el equilibrio.
- 📈 Recompensar la progresión.

---

# ⚖️ Ventajas y Desventajas

El equilibrio es uno de los pilares fundamentales de SMPES.

Por este motivo, todas las clases poseen ventajas y limitaciones.

| Tipo | Objetivo |
|------|----------|
| ✅ Ventajas | Potenciar el rol principal de la clase. |
| ❌ Desventajas | Evitar clases excesivamente poderosas. |
| 🎯 Especialización | Fomentar diferentes estilos de juego. |

No existe una clase "mejor" que otra.

Cada una destaca en determinadas situaciones y presenta debilidades que otros jugadores podrán aprovechar.

---

# 🎮 Estilos de Juego

Cada clase está orientada a un tipo de jugador diferente.

Dependiendo de la clase elegida, el jugador podrá especializarse en distintas funciones dentro del servidor.

Algunos ejemplos incluyen:

- ⚔️ Combate cuerpo a cuerpo.
- 🏹 Combate a distancia.
- 🛡️ Defensa.
- 🤝 Apoyo al equipo.
- 🧠 Estrategia y control.

La documentación individual de cada clase explica con detalle su rol y sus características.
