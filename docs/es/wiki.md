# 📖 Introducción

## ¿Qué es SMPES?

**SMPES** es un plugin desarrollado específicamente para las temporadas privadas del servidor SMPES.

Su objetivo no es reemplazar la experiencia vanilla de Minecraft, sino ampliarla mediante sistemas cuidadosamente diseñados que aporten nuevas posibilidades de juego sin perder la esencia del juego original.

Cada temporada introduce nuevas mecánicas, ajustes de equilibrio y mejoras basadas en la experiencia obtenida durante el desarrollo y las pruebas realizadas en el servidor.

---

# 🎯 Filosofía del proyecto

SMPES sigue una filosofía de desarrollo basada en la calidad antes que en la cantidad.

Cada sistema implementado debe cumplir una serie de principios fundamentales antes de formar parte de una temporada oficial.

## Principios de desarrollo

- 🎮 Mantener la esencia de Minecraft.
- ⚖️ Priorizar el equilibrio antes que la complejidad.
- 📚 Documentar todos los sistemas importantes.
- 🧪 Probar cada mecánica en un entorno real.
- 🚀 Publicar únicamente versiones estables.
- 🌍 Liberar el código fuente una vez finalizada cada temporada.

---

# 🏗️ Ciclo de desarrollo

Cada temporada sigue el mismo proceso de desarrollo.

```text
💡 Idea
   │
   ▼
💻 Desarrollo
   │
   ▼
🧪 Temporada Privada
   │
   ▼
⚖️ Balance y mejoras
   │
   ▼
📦 Release
   │
   ▼
🌍 Publicación del código
```

Este flujo garantiza que todas las funcionalidades hayan sido utilizadas y equilibradas antes de hacerse públicas.

---

# 🎮 Objetivos del proyecto

SMPES busca ofrecer una experiencia distinta sin convertir Minecraft en un juego completamente diferente.

Los principales objetivos del proyecto son:

- ⚔️ Crear sistemas con personalidad propia.
- 🏰 Fomentar el juego cooperativo mediante clanes.
- 📈 Diseñar una progresión equilibrada.
- 🛡️ Introducir contenido exclusivo sin romper el equilibrio.
- 📚 Mantener una documentación clara y actualizada.
- 💻 Compartir el desarrollo con la comunidad una vez finalizada cada temporada.
